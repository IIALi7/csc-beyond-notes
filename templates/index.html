Lost File Viewer

This system lets students read text files from the internal notes/ folder. Only safe educational notes are supposed to be accessible.

Try it

You can view a file by specifying its path using the ?file= parameter.

Example URL:
/view?file={{ example_file }}

Open example note

Custom File

Hint: The backend simply joins the file path you send with the server directory. Think about how file paths work in operating systems.
